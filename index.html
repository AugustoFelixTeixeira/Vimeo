Welcome to Java Vimeo API 3.0.

To use this api you’ll first need to register your app from Vimeo:

https://developer.vimeo.com/apps

Then you'll need to generate an Access Token with upload access. The generated Token is all you need to use the Java Vimeo API 3.0.
